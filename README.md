# 📱 Instagram Reach Analysis

A beginner data analytics project analysing personal Instagram reach and engagement data using Google Sheets and Google Looker Studio.

---

## About the Project

- **Dataset:** 17 Instagram posts and reels from November 2025 to June 2026
- **Platform:** Instagram (Professional account)
- **Tools used:** Google Sheets (data cleaning) + Google Looker Studio (interactive dashboard)
- **Project:** 3rd data analytics project

---

## Key Numbers

| Metric | Value |
|---|---|
| Total Reach | 30,002 |
| Average Engagement Rate | 16.3% |
| Best Single Post Reach | 4,219 (April 2026 — Mountains reel) |
| Best Engagement Rate | 41% (November 2025) |
| Total Posts Analysed | 17 (Reels + Photos) |

---

## Dataset Columns

| Column | Description |
|---|---|
| Date | Date the post was published |
| Month | Month name |
| Post_Type | Reel or Photo |
| Reach | Unique accounts that saw the post |
| Impressions | Total times the post was displayed |
| Likes | Total likes |
| Comments | Total comments |
| Shares | Total shares |
| Saves | Total saves |
| Engagement_Rate | (Likes+Comments+Shares+Saves) / Reach × 100 |

---

## Key Findings

- April 2026 was the best month — 8 posts, 14,695 total reach
- Mountains and nature content consistently got the highest reach
- The November post had only 605 reach but a 41% engagement rate — showing a small loyal audience
- Average engagement rate of 16.3% is nearly 5x the industry average of 3–6%
- Likes dominate engagement — 3,545 likes vs 290 shares
